# gwj90Murat
Projet Godot Wild Jam #90 à Murat


Un oiseau (ou groupe d'oiseaux) qui va d'un point de départ à un point d'arrivée.
Dans une zone semi-ouverte (couloir large) mélant plusieurs paysages successifs.
L'oiseau peut avoir des interactions (avec d'autres oiseaux, avec des nuages, ...)
Autre possibilité : migration dont le but est de perdre le moins d'oiseaux possibles

3D en Vue 3/4, au dessus de l'oiseau

Musique :
- intro avec une musique avec mélodie
- ambiances selon le paysage en dessous.
- bruitages
